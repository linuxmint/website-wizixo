Features

What is Linux Mint?

Linux Mint is an operating system designed for desktop and laptop use. Designed with 'out of the box' functionality in mind, it comes ready with applications required by the average user.

Graphic Design

Work in 3D with Blender, draw or edit pictures in Gimp, use Inkscape for vector graphics.

Productivity

LibreOffice comes preinstalled on Linux Mint, allowing you to work with standard office applications. For work, school or university – it works for everyone.

Multimedia

Watch TV, movies, listen to podcasts and music with ease using preinstalled applications, meant to give you the best user experience possible.

Web

Browse the internet, watch the content and read the content you like in Firefox. Integrate web applications into your system, enabling quick access to all your favorite websites and services.

Gaming

Play your favourite video games on Steam, install other stores to play even larger collections of video games.

Give it a try. You’ll Love it!

The majority of Linux Mint users come from Microsoft Windows and other Linux operating systems.

Why Linux Mint?

Unrivaled ease of use and workflow.

Your hardware will run faster and better than ever before. Without losing usability.

Ease of Use

Linux Mint works 'Out of the Box', configuring the system for you. The system, through its excellent application suite, can be configured to match the workflow and user interface you desire.

Simplicity

With simplicity and usability in mind, the desktop experience is fine-tuned to work for everyone. Stable and light, it works on both new and old hardware.

The Force Belongs to the Administrator

No user should feel locked out of their operating system and hardware. Linux Mint is built with user privacy and freedom in mind. Configure your operating system to work the way you like it to.

Stablity

Centralized software updates, a unique and easy to use update manager keep you up-to-date on the latest software, while keeping you secure from attackers. Linux Mint requires little maintenance.

Free and Open Source

User freedom is important to us. We strive to contribute to the open source community, basing our operating system on Debian and Ubuntu.

Community-centric

The Linux Mint community welcomes all people. A tight-knit community, which supports each user. The development team listens to the community.

Still unsure?

Interesting, new and familiar.

Both new and old users will find unrivaled ease of use. Linux Mint forums provide users with a website where they can communicate, share content and improve Linux as a whole.

Is it free?

Linux Mint is both free to use and modify. Almost all Linux Mint components are open source.

Will it work on my computer?

Linux Mint works on most computers, both modern and old. You can boot from a live USB flash drive to test the system.

Can I keep Windows?

Both Linux Mint and Windows can work on the same computer. A simple menu, when booting, allows you to choose which system you wish to use.

Will it suit my needs?

Linux Mint fits the needs of most users, with access to a large amount of open source and proprietary software, packages. Most Windows games will work on Linux Mint.
